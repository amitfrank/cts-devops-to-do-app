# cts-devops-to-do-app

## פרויקט נבנה על ידי עמית פרנק ואביב לוי קשת

## פרוייקט זה עובד על מסד נתנונים mongodb

### בשביל להריץ את כל הפרויקט יש להריץ את הפקודה:

```
docker-compose up --build -d
```

## (Nginx, Express.js, MongoDB) הפרויקט רץ על שלושה קונטיינרים המתקשרים זה עם זה 
## מגנה על האפליקצייה על ידי התמודדות עם עומס תעבורה, ומסננת בקשות זדוניות Nginx
## GET, POST, PUT, DELETE מתמודד עם בקשות NodeJS
## יושב בצד שרת ומתקשר רק איתו ולא עם המשתמש, ושומר תיעוד של כל הפעולות עליו MongoDB
